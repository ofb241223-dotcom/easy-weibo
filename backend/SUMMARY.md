# 后端实现总结

本次后端采用 `Spring Boot 3 + MyBatis + MySQL 8 + JWT + Java 17`，已完成课程项目所需的主要业务闭环。

## 已完成能力

- 认证与安全
  - 用户登录、注册、找回密码、修改密码
  - JWT 鉴权
  - BCrypt 密码哈希存储
  - 统一异常处理
  - 本地开发 CORS 配置

- 用户与资料
  - 获取当前用户
  - 用户主页信息查询
  - 关注 / 取消关注
  - 关注列表、粉丝列表、互关列表
  - 编辑昵称、简介、头像、封面图
  - 用户帖子、回复、点赞、收藏、转发列表

- 帖子与互动
  - 发帖、编辑、删除
  - 图片上传，文件存本地磁盘，数据库只存 URL
  - 点赞、收藏、转发
  - 帖子详情、评论列表、发表评论
  - 浏览量统计与浏览记录明细（作者 / 管理员可见）
  - 搜索帖子、搜索用户、标签检索
  - 隐藏帖子、举报帖子

- 聊天系统
  - 一对一私聊
  - 会话列表、未读数、最近消息预览
  - 消息发送、已读、撤回
  - 拉黑 / 解除拉黑
  - 陌生人单条开场限制，回复后可继续聊天

- 通知系统
  - 点赞、评论、关注、提及通知
  - 系统通知
  - 单条已读、全部已读

- 管理员控制台
  - 后台概览统计
  - 用户治理：封禁 / 解封、禁言 / 解除禁言
  - 内容治理：帖子撤回、帖子删除、评论删除
  - 举报处理与状态流转

## 数据持久化说明

- 图片不直接存入数据库
- 上传文件保存在本地 `uploads/`
- MySQL 中保存文件访问路径
- 点赞、评论、收藏、转发、通知、治理状态都已持久化到数据库

## 当前补充文档

- 接口文档见 [API.md](./API.md)
- 在线接口页：`http://localhost:8080/swagger-ui/index.html`
- OpenAPI JSON：`http://localhost:8080/v3/api-docs`
